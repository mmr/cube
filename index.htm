Cube
Cubo translúcido com um cubo sólido inscrito que rotaciona na tela.
Escrito em C com Glut e OpenGL.
Teclas:
TeclaDescrição
1Liga/Desliga rotação do cubo 1 (translúcido)
2Liga/Desliga rotação do cubo 2 (sólido)
q/Q/<ESC>Sai do Programa


Arquivos:
ArquivoDescrição
MakefileArquivo que facilita a compilação (para Unix/Linux)
cube.cCódigo fonte do programa
cube.exeBinário do programa pra Windows*
cube (com timer)Binário do programa pra MacOS X
cube_(sem timer)Binário do programa pra MacOS X
glut32.dllGlut para Windows

* Você precisa do glut32.dll para rodar o binário.
Basta copiá-lo para %SystemRoot%\system32 (WinXP/2K) ou %windir%\system (Win9x).
O binário foi criado com o MingW 3.2.3 em plataforma Windows XP (2600sp1).
Para compilá-lo usando MingW, use:
gcc cube.c -o cube -Lc:\mingw\lib -lopengl32 -lglu32 -lglut32
